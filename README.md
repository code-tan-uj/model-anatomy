# model-anatomy

> A structured, ground-up journey through AI engineering — from mathematical foundations to building and reproducing landmark research.

This repository is a living study journal. The goal is not just to read papers but to **understand, implement, and reproduce** the ideas from first principles.

---

## What lives here

| Directory | Contents |
|-----------|----------|
| `notes/` | Concept breakdowns, paper summaries, and my own explanations |
| `implementations/` | From-scratch code implementations of key architectures |
| `experiments/` | Jupyter notebooks for exploration and ablation studies |
| `resources/` | Curated free resources, links, and reading lists |
| `ROADMAP.md` | The full learning plan — phases, papers, and milestones |

> **Note:** Research papers and textbooks are stored **locally only** and are excluded from version control (`.gitignore`). All paper references include official sources, arXiv links, and publisher links so you can access them legally.

---

## Philosophy

- **Build, don't just read.** Every major concept gets a from-scratch implementation.
- **Understand the math.** No black boxes. Derive it, then code it.
- **Reproduce before extending.** Reproduce key results before building on top of them.
- **Document everything.** The `notes/` directory is as important as the code.

---

## Roadmap Overview

The journey is split into 7 phases:

```
Phase 0  →  Mathematical Foundations
Phase 1  →  Classical Machine Learning
Phase 2  →  Deep Learning & Neural Networks
Phase 3  →  The Transformer Revolution
Phase 4  →  Large Language Models
Phase 5  →  Applied LLM Engineering (RAG, RLHF, Fine-tuning)
Phase 6  →  Agentic AI & Tool Use
Phase 7  →  Production, Safety & LLMOps
```

See [ROADMAP.md](./ROADMAP.md) for the full detailed plan with papers, resources, and implementation targets.
